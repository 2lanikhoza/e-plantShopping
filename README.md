\# Paradise Nursery Shopping Application



Paradise Nursery is an online plant shopping cart application built with React. The project allows users to browse different houseplants, view plant details such as images, names, descriptions, and prices, and add selected plants to a shopping cart.



\## Project Name



Paradise Nursery Shopping Application



\## Project Description



This application was created as a final project to demonstrate React components, routing, Redux state management, and shopping cart functionality. Users can navigate between the landing page, product listing page, and cart page. The cart updates dynamically when users add, increase, decrease, or remove plants.



\## Features



\- Landing page with a “Get Started” button

\- Navigation bar with links to Home, Plants, and Cart

\- Product listing page with houseplants grouped into categories

\- Plant cards with image, name, description, price, and Add to Cart button

\- Shopping cart page showing selected plants

\- Increase and decrease quantity buttons

\- Delete item button

\- Continue Shopping button

\- Checkout button showing a coming soon message

\- Dynamic cart item count
